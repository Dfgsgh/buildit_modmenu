# buildit_modmenu
a simple modmenu for r/buildit

# how to install

## step 1 -  get a userscript manager extension

here's a compatibility table i deffinitely didn't steal

| Browser | Userscript manager |
|--------|--------------------|
| Chrome, dev builds or Chromium-based browsers (Windows, macOS, Linux) | Tampermonkey, Violentmonkey, ScriptCat or OrangeMonkey |
| Safari (macOS, iOS, iPadOS, visionOS) | Stay or Userscripts |
| Edge or dev builds (Windows, macOS, Linux) | Tampermonkey, Violentmonkey, ScriptCat or OrangeMonkey |
| Edge or dev builds (Android) | Tampermonkey, Violentmonkey or ScriptCat (install from => Extensions) |
| Firefox, dev builds or Gecko-based browsers (Windows, macOS, Linux) | Tampermonkey, Violentmonkey or ScriptCat |
| Firefox, dev builds or Gecko-based browsers (Android) | Tampermonkey, Violentmonkey or ScriptCat |
| QQ Browser (Windows, macOS) | Not required (Tampermonkey & Violentmonkey pre-installed) |
| Mises (Android, iOS, iPadOS) | Tampermonkey or ScriptCat |
| Lemur (Android, iOS) | ScriptCat |
| Quetta (Android, iOS, iPadOS) | ScriptCat |
| Orion (iOS, iPadOS) | Tampermonkey, ScriptCat or OrangeMonkey |

after you've installed the browser extension proceed to step 2

## step 2 - copy the userscript

in the repository access the userscript.js file and copy it to clipboard

## step 3 - add the userscript 

access the userscript manager's menu and figure out how to add a new script, in tampermonkey you click "create new script" and paste the coppied text, that's it enjoy the modmenu


# Features

the modmenu currently ads 4 features:

- a change name button
- a force clear button
- a force bublish button
- save/load buttons

the modmenu is currently editor-oriented but that can change soon
